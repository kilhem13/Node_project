## Introduction

The basic Node.js application on TypeScript.

## Installing

```bash
git clone https://github.com/sergkudinov/ece-nodejs-3-leveldb.git
cd ece-nodejs-3-leveldb
npm install
```

## Build

```bash
npm run build
```

## Development

```bash
npm run dev
```
